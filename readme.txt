To use the system, you first need to type in username and password to get oracle connection, then the table warehouses, distribution_station, customers, orders, lineItems, items and warehouse_stock will be generated automatically. 

Then the system will ask you to type in the number of distribution stations, the number of 
customers, the number of items, the maximum number for each customers, and the maximum number of orders for each customer. After typing all of these information, data will be inserted to the tables automatically.
